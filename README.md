## 基本操作

- 打开或关闭背包：B按键
- 界面交互：鼠标点击
- 退出：Q按键

## 背包设定

- 背包格子上限为30
- 单个物品存在最大堆叠上限，不同物品堆叠上限不同
- 物品之间按编写的设定进行排序摆放
- 制作物品数量最少不可小于1，最大不可超过背包格子上限

## 使用插件

- [SQLite4Unity3d](https://github.com/robertohuertasm/SQLite4Unity3d)：用于将unity与SQlite进行连接
